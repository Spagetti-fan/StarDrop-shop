# ★ StarDrop — цифровой магазин

Telegram Mini App + сайт для продажи цифровых товаров.

🔗 **Живой сайт:** https://sunny-dusk-7fe4ed.netlify.app/

## Что реализовано

- 🪼 Виртуальный помощник (анимированная медуза)
- 🛒 Корзина с промокодами и скидками
- ✈️ Telegram Mini App (BackButton, HapticFeedback, MainButton)
- 📬 Уведомления о заказах через Cloudflare Worker
- 🎆 Конфетти и частицы при оформлении заказа
- ⚙️ Настройки (анимации, звуки, уведомления)
- 📱 6 страниц: Главная, Telegram, Roblox, Разработка, Поддержка, Настройки

## Технологии

HTML · CSS · JavaScript · Telegram WebApp API · Cloudflare Workers · Netlify

## Товары

- Telegram Stars, Premium, NFT
- Roblox Robux, донат, вещи
- Разработка ботов и сайтов
